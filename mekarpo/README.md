## Installation

1. Clone the repo.
   ```sh
   git clone https://github.com/HelgaBK/CourseWork_1st_mag.git
   ```
2. Use npm install to install the dependencies.
    ```sh
    npm install
    ```

## Usage

Use npm start to start development mode.
```sh
npm start
```
Use npm build to generate the production bundle.
```sh
npm build
```
